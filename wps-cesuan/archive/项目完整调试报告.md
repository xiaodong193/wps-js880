## 一、项目概述

### 1.1 项目简介

WPS租金测算系统V2是一个基于WPS Office JSA（JavaScript API）的专业财务计算系统，用于租赁业务的租金测算、现金流量表生成等核心功能。系统采用配置驱动架构，具有高度的可维护性和扩展性。

### 1.2 技术架构

- **运行环境**: WPS Office + JSA (JavaScript API)
- **编程语言**: JavaScript (ES6+)
- **架构模式**: 配置驱动架构 + 模块化设计
- **设计原则**: 单一职责、开闭原则、依赖倒置
- **命名规范**: cls前缀类名 + 驼峰命名法 + 中文函数支持

### 1.3 核心特性

1. ✅ **配置驱动架构**: 所有配置集中在CONFIG对象中，修改配置无需改动业务逻辑
2. ✅ **零重复代码**: 提取所有公共逻辑，统一错误处理机制
3. ✅ **模块化设计**: 清晰的职责划分，易于维护和扩展
4. ✅ **向后兼容**: 保留V1版本的函数接口
5. ✅ **完善的错误处理**: 统一的错误处理和日志记录
6. ✅ **R1C1公式样式**: 使用相对引用公式，可复制粘贴
7. ✅ **三期模式设计**: 区分首期、中间期、末期的不同计算逻辑

#### 3.1.2 动态属性生成

**示例代码**:
```javascript
class clsParameterManager {
    _createParameterAccessors() {
        const paramNames = this.GetParameterNames();
        paramNames.forEach(name => {
            // 动态生成 R1C1 属性
            Object.defineProperty(this, `${name}CellR1C1`, {
                get: () => this.GetConfigValue(name, "CellAddress"),
                configurable: true
            });
            
            // 动态生成 A1 属性
            Object.defineProperty(this, `${name}CellA1`, {
                get: () => this.GetCellAddressA1(name),
                configurable: true
            });
            
            // 动态生成 Value 属性
            Object.defineProperty(this, `${name}CellValue`, {
                get: () => this.ReadParameterValue(name),
                configurable: true
            });
        });
    }
}
```

**技术解析**:
1. **Object.defineProperty**: 动态定义对象属性
2. **getter函数**: 通过getter访问，每次调用都执行函数
3. **configurable**: 允许属性被删除或修改

#### 3.1.3 三元运算符嵌套

**示例代码**:
```javascript
arrToArrData(arrFormula) {
    for (let row = 0; row < totalPeriodsValue; row++) {
        // 使用三元运算符确定公式类型
        const rowIndex = row === 0 ? 1 : (row === totalPeriodsValue - 1 ? 3 : 2);
        // 
        // 解释：
        // - row === 0 ? 1 : ...  → 第1行返回1（首期公式）
        // - row === totalPeriodsValue - 1 ? 3 : 2  → 最后一行返回3（末期公式），其他返回2（中间期公式）
        //
        // 等价于：
        // if (row === 0) {
        //     rowIndex = 1;
        // } else if (row === totalPeriodsValue - 1) {
        //     rowIndex = 3;
        // } else {
        //     rowIndex = 2;
        // }
        
        arrData[row][col - 1] = arrFormula[rowIndex][col];
    }
}
```

**技术解析**:
1. **三元运算符**: 条件 ? 值1 : 值2
2. **嵌套三元运算符**: 条件1 ? 值1 : (条件2 ? 值2 : 值3)
3. **优先级**: 三元运算符从右向左计算

**设计优势**:
- ✅ 代码简洁：一行代码替代多行if-else
- ✅ 可读性好：熟练后易于理解
- ✅ 性能优良：编译器优化良好

**注意事项**:
- ⚠️ 过度嵌套会降低可读性，建议最多嵌套2-3层
- ⚠️ 复杂逻辑建议使用if-else或switch

#### 3.1.4 R1C1公式样式

**示例代码**:
```javascript
// A1样式公式（绝对引用，复制后需要调整）
arr[1][3] = "=ROUND(-PMT($B$5/$B$11,$B$10,$B$4,0),2)";
arr[2][3] = "=ROUND(-PMT($B$5/$B$11,$B$10,$B$4,0),2)";

// R1C1样式公式（相对引用，可复制粘贴）
arr[1][3] = `=ROUND(-PMT(${interestRateCell}/R11C2,${totalPeriodsCell},${principalCell},0),2)`;
arr[2][3] = arr[1][3];  // 直接复制，无需修改
arr[2][1] = "=R[-1]C+1";  // 相对引用：上一行+1
```

**技术解析**:
1. **R1C1引用格式**:
   - `R5C3` = 第5行第3列（绝对引用）
   - `R[-1]C` = 上一行同列（相对引用）
   - `RC[1]` = 同行下一列（相对引用）
   
2. **优势**:
   - 可复制粘贴到任何位置，公式自动适应
   - 适合批量生成相同模式的公式

3. **转换方法**:
   ```javascript
   // R1C1转A1
   worksheet.Cells(5, 3).Address(false, false);  // 返回 "C5"
   
   // A1转R1C1
   worksheet.Range("C5").Address(true, true, 1);  // 返回 "R5C3"
   ```

**设计优势**:
- ✅ 公式可移植：复制到其他位置无需修改
- ✅ 减少错误：避免手动调整引用
- ✅ 提高效率：批量生成公式

#### 3.2.4 三期模式

**概念**:
- **首期**: 第1期，有特殊的起始计算
- **中间期**: 第2期到倒数第2期，使用相对引用公式
- **末期**: 最后一期，需要调整确保总和正确

**实现方式**:
```javascript

for (let i = 0; i < 5; i++) {
    arr[i] = new Array(13);
}

// 第1行（索引1）= 首期公式
arr[1][1] = "1";  // 期次
arr[1][2] = `=EDATE(${leaseStartDateCell}, ${paymentInterval})`;  // 支付日
arr[1][3] = `=ROUND(-PMT(...),2)`;  // 租金
// ...

// 第2行（索引2）= 中间期公式
arr[2][1] = "=R[-1]C+1";  // 期次递增
arr[2][2] = "=EDATE(R[-1]C, " + paymentInterval + ")";  // 支付日递增
arr[2][3] = arr[1][3];  // 租金公式相同（使用R1C1相对引用）
// ...

// 第3行（索引3）= 末期公式
arr[3][1] = arr[2][1];  // 期次公式相同
arr[3][2] = "=EDATE(" + leaseStartDateCell + "," + paymentInterval + "*" + totalPeriodsValue + ")";  // 最后支付日
arr[3][4] = `=${principalCell} - SUM(R[${-totalPeriodsValue + 1}]C:R[-1]C)`;  // 最后一期本金调整
// ...

// 转换为实际数据
arrToArrData(arrFormula) {
    for (let row = 0; row < totalPeriodsValue; row++) {
        // 确定公式类型
        const rowIndex = row === 0 ? 1 : (row === totalPeriodsValue - 1 ? 3 : 2);
        
        // 复制对应类型的公式
        for (let col = 1; col <= maxCol; col++) {
            arrData[row][col - 1] = arrFormula[rowIndex][col];
        }
    }
}
```

**设计优势**:
- ✅ 区分不同期次的计算逻辑
- ✅ 最后一期可以单独调整（确保总和正确）
- ✅ 中间期公式可以复制粘贴（使用R1C1相对引用）
- ✅ 提高公式准确性

```javascript
     * @param {string} moduleName - 模块名称

    static handle(moduleName, error, level = "ERROR", showAlert = true) {
        const errorMessage = error instanceof Error ? error.message : String(error);

                break;
            case "CRITICAL":
                console.error(`[${timestamp}] [${moduleName}] 致命错误: ${errorMessage}`);
                console.error(`[${moduleName}] 错误堆栈:`, error.stack);
                if (showAlert) {
                    alert(`致命错误：${errorMessage}\n\n请立即联系技术支持！`);
                }
                break;
        }
        
        return error;
    }
    
    /**
     * 安全执行函数
     * @param {string} moduleName - 模块名称
     * @param {Function} fn - 要执行的函数
     * @param {*} defaultValue - 发生错误时返回的默认值
     * @param {boolean} showAlert - 是否显示警告框
     * @returns {*} 函数执行结果或默认值
     */
    static safeExecute(moduleName, fn, defaultValue = null, showAlert = false) {
        try {
            return fn();
        } catch (error) {
            this.handle(moduleName, error, "ERROR", showAlert);
            return defaultValue;
        }
    }
}
```

## 六、操作回退功能设计

### 6.1 功能需求

1. **记录操作历史**: 记录所有修改工作表的操作
2. **支持撤销**: 可以撤销最近的操作
3. **支持重做**: 可以重做已撤销的操作
4. **历史管理**: 限制历史记录数量（如最多100条）
5. **状态保存**: 在关键操作前后保存工作表状态

### 6.2 架构设计

```javascript
/**
 * ============== 操作历史管理器 ==============
 * 功能：记录和管理操作历史，支持撤销和重做
 */
class clsOperationHistory {
    constructor() {
        this.MODULE_NAME = "clsOperationHistory";
        this.MAX_HISTORY_SIZE = 100;  // 最大历史记录数
        this.historyStack = [];       // 历史记录栈
        this.currentPosition = -1;    // 当前位置指针
        this.isEnabled = true;         // 是否启用历史记录
    }
    
    /**
     * 记录操作
     * @param {string} operationType - 操作类型（insert/delete/update等）
     * @param {string} sheetName - 工作表名称
     * @param {Object} beforeState - 操作前状态
     * @param {Object} afterState - 操作后状态
     * @param {Object} metadata - 元数据（位置、参数等）
     */
    recordOperation(operationType, sheetName, beforeState, afterState, metadata = {}) {
        if (!this.isEnabled) {
            console.log(`[${this.MODULE_NAME}] 历史记录已禁用，跳过记录`);
            return;
        }
        
        const operation = {
            timestamp: new Date().toISOString(),
            type: operationType,
            sheetName: sheetName,
            before: beforeState,
            after: afterState,
            metadata: metadata,
            description: this.generateDescription(operationType, metadata)
        };
        
        // 如果当前不在栈顶，删除当前位置之后的所有记录
        if (this.currentPosition < this.historyStack.length - 1) {
            this.historyStack = this.historyStack.slice(0, this.currentPosition + 1);
        }
        
        // 添加新记录
        this.historyStack.push(operation);
        this.currentPosition++;
        
        // 限制历史记录数量
        if (this.historyStack.length > this.MAX_HISTORY_SIZE) {
            this.historyStack.shift();
            this.currentPosition--;
        }
        
        console.log(`[${this.MODULE_NAME}] 记录操作: ${operation.description}`);
        console.log(`[${this.MODULE_NAME}] 历史记录: ${this.currentPosition + 1}/${this.historyStack.length}`);
    }
    
    /**
     * 撤销操作
     * @returns {Object} 撤销结果
     */
    undo() {
        if (!this.canUndo()) {
            console.log(`[${this.MODULE_NAME}] 无法撤销：没有可撤销的操作`);
            return { success: false, message: "没有可撤销的操作" };
        }
        
        const operation = this.historyStack[this.currentPosition];
        console.log(`[${this.MODULE_NAME}] 撤销操作: ${operation.description}`);
        
        try {
            // 恢复操作前状态
            this.restoreState(operation.before);
            
            this.currentPosition--;
            console.log(`[${this.MODULE_NAME}] 撤销成功`);
            
            return {
                success: true,
                operation: operation,
                message: "撤销成功"
            };
        } catch (error) {
            console.error(`[${this.MODULE_NAME}] 撤销失败: ${error.message}`);
            return {
                success: false,
                operation: operation,
                message: `撤销失败: ${error.message}`
            };
        }
    }
    
    /**
     * 重做操作
     * @returns {Object} 重做结果
     */
    redo() {
        if (!this.canRedo()) {
            console.log(`[${this.MODULE_NAME}] 无法重做：没有可重做的操作`);
            return { success: false, message: "没有可重做的操作" };
        }
        
        this.currentPosition++;
        const operation = this.historyStack[this.currentPosition];
        console.log(`[${this.MODULE_NAME}] 重做操作: ${operation.description}`);
        
        try {
            // 恢复操作后状态
            this.restoreState(operation.after);
            
            console.log(`[${this.MODULE_NAME}] 重做成功`);
            
            return {
                success: true,
                operation: operation,
                message: "重做成功"
            };
        } catch (error) {
            console.error(`[${this.MODULE_NAME}] 重做失败: ${error.message}`);
            this.currentPosition--;  // 回退指针
            return {
                success: false,
                operation: operation,
                message: `重做失败: ${error.message}`
            };
        }
    }
    
    /**
     * 恢复状态
     * @param {Object} state - 要恢复的状态
     */
    restoreState(state) {
        // 获取工作表
        const worksheet = Application.Worksheets(state.sheetName);
        
        // 恢复数据
        if (state.data) {
            const range = worksheet.Range(state.range);
            range.Value2 = state.data;
        }
        
        // 恢复参数
        if (state.parameters) {
            for (const [paramName, value] of Object.entries(state.parameters)) {
                const cellAddress = p.addr(paramName);
                if (cellAddress) {
                    worksheet.Range(cellAddress).Value2 = value;
                }
            }
        }
        
        // 触发重新计算
        worksheet.Calculate();
    }
    
    /**
     * 捕获工作表状态
     * @param {string} sheetName - 工作表名称
     * @param {string} range - 数据范围
     * @returns {Object} 工作表状态对象
     */
    captureState(sheetName, range) {
        const worksheet = Application.Worksheets(sheetName);
        
        return {
            sheetName: sheetName,
            range: range,
            data: worksheet.Range(range).Value2,
            parameters: {
                TotalPeriods: p.val("TotalPeriods"),
                Principal: p.val("Principal"),
                InterestRate: p.val("InterestRate"),
                // ... 捕获更多关键参数
            }
        };
    }
    
    /**
     * 生成操作描述
     * @param {string} operationType - 操作类型
     * @param {Object} metadata - 元数据
     * @returns {string} 操作描述
     */
    generateDescription(operationType, metadata) {
        const descriptions = {
            'insert': `在位置 ${metadata.position} 插入 ${metadata.rowCount} 行`,
            'delete': `删除位置 ${metadata.position} 的 ${metadata.rowCount} 行`,
            'update': `更新数据范围 ${metadata.range}`,
            'clear': `清除工作表 ${metadata.sheetName} 数据`
        };
        
        return descriptions[operationType] || `未知操作: ${operationType}`;
    }
    
    /**
     * 检查是否可以撤销
     * @returns {boolean}
     */
    canUndo() {
        return this.currentPosition >= 0;
    }
    
    /**
     * 检查是否可以重做
     * @returns {boolean}
     */
    canRedo() {
        return this.currentPosition < this.historyStack.length - 1;
    }
    
    /**
     * 清除历史记录
     */
    clearHistory() {
        this.historyStack = [];
        this.currentPosition = -1;
        console.log(`[${this.MODULE_NAME}] 历史记录已清除`);
    }
    
    /**
     * 获取历史记录列表
     * @returns {Array} 历史记录数组
     */
    getHistoryList() {
        return this.historyStack.map((op, index) => ({
            index: index,
            timestamp: op.timestamp,
            description: op.description,
            isCurrent: index === this.currentPosition
        }));
    }
    
    /**
     * 启用/禁用历史记录
     * @param {boolean} enabled - 是否启用
     */
    setEnabled(enabled) {
        this.isEnabled = enabled;
        console.log(`[${this.MODULE_NAME}] 历史记录已${enabled ? '启用' : '禁用'}`);
    }
}
```
