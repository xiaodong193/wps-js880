## 四、兼容性转换示例

### 4.1 可选链操作符转换表

| 原语法 (ES2020+) | 兼容语法 (ES6-ES2019) | 说明 |
|-----------------|---------------------|------|
| `obj?.prop` | `obj && obj.prop` | 属性访问 |
| `obj?.[key]` | `obj && obj[key]` | 动态属性访问 |
| `arr?.[index]` | `arr && arr[index]` | 数组索引访问 |
| `func?.()` | `func && func()` | 函数调用 |

### 4.2 空值合并运算符转换表

| 原语法 (ES2020+) | 兼容语法 (ES6-ES2019) | 说明 |
|-----------------|---------------------|------|
| `a ?? b` | `(a !== undefined && a !== null) ? a : b` | 空值合并 |
| `a ?? default` | `a != null ? a : default` | 简写形式 |

### 4.3 实际代码转换示例

**示例1: 安全获取嵌套属性**
```javascript
// ES2020+ 语法
const value = config?.settings?.rate ?? 0.05;

// ES6-ES2019 兼容语法
const value = (config && config.settings && config.settings.rate != null) 
    ? config.settings.rate 
    : 0.05;
```

**示例2: 安全调用方法**
```javascript
// ES2020+ 语法
const result = obj?.calculate?.(param) ?? 0;

// ES6-ES2019 兼容语法
const result = (obj && obj.calculate && typeof obj.calculate === 'function')
    ? obj.calculate(param)
    : 0;
```

**示例3: 数组安全访问**
```javascript
// ES2020+ 语法
const item = arr?.[0]?.name ?? "Unknown";

// ES6-ES2019 兼容语法
const item = (arr && arr[0] && arr[0].name != null) 
    ? arr[0].name 
    : "Unknown";
```

## 六、建议与注意事项

### 6.1 开发建议

1. **编码时使用ESLint**: 配置ESLint规则，禁止使用ES2020+语法
2. **代码审查**: 提交前进行语法兼容性检查
3. **自动化测试**: 在WPS环境中测试所有功能

### 6.2 常见陷阱

1. **注意?.和??的区别**: 有时?.和||组合也等同于??，需要一并检查
2. **数组索引**: `arr?.[index]` 也需要转换
3. **函数调用**: `func?.()` 也需要转换
