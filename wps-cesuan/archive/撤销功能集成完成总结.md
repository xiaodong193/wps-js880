## 三、使用方式

### 3.1 方式一：全局管理器（推荐简单场景）

```javascript
// 加载 mUndoManager_v2.js 后自动创建全局实例

// 在任何地方使用全局撤销
const result = globalUndo();
if (result.success) {
    console.log("已撤销:", result.message);
}

// 重做
globalRedo();

// 查看状态
showUndoStatus();
```

### 3.2 方式二：独立管理器（推荐复杂场景）

```javascript
// 为特定模块创建独立管理器
const undoManager = createUndoManager({
    maxHistory: 50,
    enableGrouping: true
});

// 创建调息实例并指定管理器
const calc = new clsInterestRateAdjustment(null, undoManager);
```

### 3.3 方式三：命令模式（推荐精细控制）

```javascript
// 创建自定义命令
const command = new clsWorksheetModifyCommand(
    worksheet,
    "A1:B10",
    oldData,
    newData,
    "修改区域A1:B10"
);

// 通过管理器执行
getUndoManager().execute(command);
```

    calc.addAdjustmentWithUndoManager(12, 0.04);

    showUndoHistoryUnified();
    
    // 5. 撤销
    globalUndo();
    
    // 6. 显示撤销后的状态
