## 一、项目概述

### 1.1 项目简介

WPS租金测算系统V2是一个基于WPS Office JSA（JavaScript API）的财务计算系统，用于租赁业务的租金测算、现金流量表生成等核心功能。

### 1.2 技术栈

- **运行环境**: WPS Office + JSA (JavaScript API)
- **编程语言**: JavaScript (ES6+)
- **架构模式**: 配置驱动架构 + 模块化设计
- **命名规范**: cls前缀类名 + 驼峰命名法

### 1.3 核心特性

1. ✅ **配置驱动架构**: 所有配置集中在CONFIG对象中
2. ✅ **零重复代码**: 提取公共逻辑，统一错误处理
3. ✅ **模块化设计**: 清晰的职责划分
4. ✅ **向后兼容**: 保留V1版本的函数接口

##### 等额租金法arr() - 等额租金公式生成

**功能**: 生成等额租金法（等额本息后付）的R1C1公式数组

**公式逻辑**:

**第1期**:
```javascript
arr[1][1] = "1";  // 期次
arr[1][2] = "=EDATE(起租日, 支付间隔)";  // 支付日
arr[1][3] = "=ROUND(-PMT(利率/期利率, 总期数, 本金, 0), 2)";  // 租金
arr[1][4] = "=ROUND(-PPMT(利率/期利率, 期次, 总期数, 本金, 0), 2)";  // 本金
arr[1][5] = "=租金-本金";  // 利息
arr[1][6] = "=本金";  // 累积偿还本金
arr[1][7] = "=总成本-累积本金";  // 租金本金余额
arr[1][8] = "=SUM(所有期租金)-已还租金";  // 剩余租金余额
```

**第2期-倒数第2期**:
```javascript
arr[2][1] = "=R[-1]C+1";  // 期次递增
arr[2][2] = "=EDATE(上期支付日, 支付间隔)";  // 支付日递增
arr[2][3] = arr[1][3];  // 租金公式相同（使用R1C1相对引用）
arr[2][4] = arr[1][4];  // 本金公式相同
arr[2][5] = arr[1][5];  // 利息公式相同
arr[2][6] = "=本期本金+上期累积本金";  // 累积本金累加
arr[2][9] = "=本期租金+上期已还租金";  // 已还租金累加
```

**倒数第1期（最后一期）**:
```javascript
arr[3][1] = arr[2][1];  // 期次公式相同
arr[3][2] = "=EDATE(起租日, 支付间隔*总期数)";  // 最后支付日
arr[3][3] = arr[1][3];  // 租金公式相同
arr[3][4] = "=总成本-SUM(第1期到倒数第2期本金)";  // 最后一期本金调整
arr[3][6] = arr[2][6];  // 累积本金公式相同
arr[3][9] = "=本期租金+上期已还租金";  // 已还租金累加
```

**技术亮点**:
- **R1C1样式**: 使用相对引用，公式可以复制粘贴
- **三期模式**: 区分首期、中间期、末期的不同公式
- **Excel函数**: 使用PMT、PPMT等财务函数
- **ROUND函数**: 确保金额精确到小数点后2位

##### 等额本金法按天计息arr() - 等额本金按天计息公式

**功能**: 生成等额本金法按天计息的公式数组

**核心特点**: 每期本金相同，利息按实际天数计算

**第1期公式**:
```javascript
arr[1][1] = "1";
arr[1][2] = "=EDATE(起租日, 支付间隔)";
arr[1][3] = "=ROUND(本金+利息, 2)";
arr[1][4] = "=ROUND(总成本/总期数, 2)";  // 每期本金相同
arr[1][5] = "=ROUND(总成本*利率/360*(本期支付日-起租日), 2)";  // 按天计息
arr[1][6] = "=本金";  // 累积本金
```

**第2期-倒数第2期**:
```javascript
arr[2][1] = "=R[-1]C+1";
arr[2][2] = "=EDATE(上期支付日, 支付间隔)";
arr[2][4] = arr[1][4];  // 每期本金相同
arr[2][5] = "=ROUND(上期余额*利率/360*(本期支付日-上期支付日), 2)";
```

**倒数第1期（关键修改点）**:
```javascript
// 修改前（需要调整）:
arr[3][4] = "=总成本-SUM(第1期到倒数第2期本金)";

// 修改后（保持一致）:
arr[3][4] = arr[1][4];  // 与第1期本金公式相同
```

**技术亮点**:
- **等额本金**: 每期本金完全相同 = 总成本/总期数
- **按天计息**: 利息 = 余额 × 利率/360 × 实际天数
- **360天制**: 使用360天而不是365天计算年利率

##### 本金比例法arr() - 本金比例法公式

**功能**: 根据本金比例计算每期偿还金额

**特点**:
- 每期本金比例可自定义
- 第1期-倒数第2期：相同比例
- 最后一期：调整比例确保总和为100%

```javascript
// 第1期本金比例
arr[1][12] = "=round(100/总期数, 2)";  // 平均分配

// 最后一期本金比例（调整）
arr[3][12] = "=100-SUM(第1期到倒数第2期比例)";  // 确保总和100%
```
