## 🚀 5分钟快速开始

### 第一步：准备数据
在 WPS 表格中准备数据：
```
产品    年份    销售额
A       2023    100
A       2024    200
B       2023    300
B       2024    400
```

### 第二步：运行代码
按 `Alt+F11` 打开宏编辑器，粘贴：

```javascript
function 快速测试() {
    var ws = Application.ActiveWorkbook.Worksheets("Sheet1");
    var data = ws.Range("A1:C5").Value2;
    
    var result = Array2D.z超级透视(
        data,
        ['f1', '产品'],
        ['f2', '年份'],
        ['sum("f3")', '销售额']
    );
    
    var newWs = Application.ActiveWorkbook.Worksheets.Add();
    newWs.Name = "透视结果";
    result.toRange("A1");
    
    Console.log("完成！");
}

快速测试();
```

### 第三步：查看结果
在「透视结果」工作表中查看生成的透视表。
