## 🔗 文件依赖关系

### superPivot 功能链

```
用户代码
    │
    ├── → Array2D.z超级透视() ── → 生成透视表数据
    │                              │
    │                              ├── → 内部: 构建表头
    │                              │
    │                              └── → 内部: 数据聚合
    │
    ├── → result.toRange() ────── → 输出到工作表
    │
    └── → RngUtils.mergeCells() ─ → 合并标题单元格
                                    (策略: "cm")
```
