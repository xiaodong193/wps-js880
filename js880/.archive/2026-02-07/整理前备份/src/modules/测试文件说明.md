## 🛠️ 常见问题

### Q1: 运行测试时提示 "Array2D 未定义"

**原因**: 未加载 JSA880.js 框架

**解决**: 确保 JSA880.js 已导入或 superPivot 函数已定义

### Q2: 测试通过但没有生成工作表

**原因**: WPS 工作簿可能被保护或只读

**解决**: 
1. 确保工作簿可编辑
2. 检查是否有同名工作表被保护

### Q3: 百分比测试显示不正确

**原因**: 测试数据随机生成，结果不固定

**解决**: 这是正常现象，只要没有报错即表示功能正常

### Q4: 性能测试耗时过长

**原因**: 大数据量测试（1000行）在某些机器上较慢

**解决**: 
1. 这是正常的，取决于机器性能
2. 如超过10秒，可能需要优化代码

## 📝 自定义测试

### 使用自己的数据测试

```javascript
function 测试我的数据() {
    // 从工作表读取数据
    var ws = Application.ActiveWorkbook.Worksheets("我的数据");
    var data = ws.Range("A1:D100").Value2;
    
    // 创建透视表
    var result = Array2D.z超级透视(
        data,
        ['f1', '分类'],
        ['f2', '时间'],
        ['sum("f4")', '数值']
    );
    
    // 输出到新工作表
    var newWs = Application.ActiveWorkbook.Worksheets.Add();
    newWs.Name = "我的透视表";
    result.toRange("A1");
    
    Console.log("完成！");
}

测试我的数据();
```
