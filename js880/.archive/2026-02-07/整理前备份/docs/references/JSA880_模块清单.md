## 概述

JSA880 是一个专为 WPS Office JavaScript API (JSA) 设计的快速开发框架，采用零依赖设计，仅使用 WPS 原生对象和标准 JavaScript。

**核心特性**:
- 零依赖设计，仅使用 WPS 原生对象
- 完整的 ES6+ 支持（let/const、箭头函数、class、async/await）
- 链式调用 API 设计
- 双语方法名（中文/英文）

### 2. SuperPivot - 超级透视表 ⭐核心功能

**代码位置**: JSA880.js:5900-7223

**功能描述**: 实现类似 Excel 数据透视表的功能，支持多级行/列标题、多种聚合函数、自动合并单元格。

**静态方法签名**:
```javascript
Array2D.z超级透视(
    data,           // Array2D 对象或二维数组
    rowFields,      // 行字段配置 ['f1,f2', '标题1,标题2']
    colFields,      // 列字段配置 ['f3', '标题3']
    dataFields,     // 数据字段 ['count(),sum("f4")', '统计,求和']
    headerRows,     // 数据源表头行数 (默认 0)
    outputHeader    // 输出表头模式: 1=完整, 0=无, -1=不含行标题, -2=不生成
)
```

**支持的聚合函数**:
- `count()` - 计数
- `sum("字段")` - 求和
- `average("字段")` - 平均值
- `min("字段")` - 最小值
- `max("字段")` - 最大值
- `textjoin("分隔符")` - 文本连接

**排序指示符**:
- `f1+` - 升序
- `f1-` - 降序
- `f1#` - 保持原顺序

**表头输出模式**:
| 值 | 说明 | 示例 |
|----|------|------|
| `1` | 完整表头（含行标题和列标题） | 3行表头 |
| `0` | 无表头 | 仅数据 |
| `-1` | 表头不含行标题 | 2行表头 |
| `-2` | 不生成表头行 | 仅数据 |

**合并单元格支持**:
- 单列字段：自动合并相同值的列
- 多列字段：每层级独立合并
- 使用 `RngUtils.mergeCells()` 实现

## 已知问题与解决方案

### 1. SuperPivot 原型方法未定义
**问题**: `Array2D.prototype.z超级透视` 在 JSA 中未定义
**解决方案**: 必须使用静态方法 `Array2D.z超级透视(data, ...)`

### 2. RngUtils 可能未定义
**问题**: `RngUtils` 在某些加载顺序下可能未定义
**解决方案**: 使用前检查 `typeof RngUtils !== 'undefined'`

### 3. SuperMap 方法缺失
**问题**: `SuperMap.fromObject` 和 `SuperMap.fromArray` 未定义
**解决方案**: 使用 `new SuperMap()` 或手动转换
