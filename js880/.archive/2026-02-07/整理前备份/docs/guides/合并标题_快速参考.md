## 📋 合并策略速查

| 策略 | 用途 | 示例 |
|------|------|------|
| `"cm"` | **透视表专用**（推荐） | `mergeCells(rng, "cm")` |
| `"c"` | 纵向合并（按列） | `mergeCells(rng, "c")` |
| `"r"` | 横向合并（按行） | `mergeCells(rng, "r")` |
| `"rm"` | 先行后列 | `mergeCells(rng, "rm")` |

## 🔧 常用操作

### 合并透视表标题
```javascript
// 计算标题行数 = 列字段数 + 1
var titleRowCount = 3  // 例如：2个列字段 + 1
var titleRng = Range("A1").Resize(titleRowCount, result[0].length)
RngUtils.mergeCells(titleRng, "cm")
```

### 取消合并并保留数据
```javascript
RngUtils.unmergeCells(titleRng, true)  // true = 填充所有单元格
```

### 快捷方式
```javascript
// 全局快捷方式
$.mergeCells("A1:G10", "cm")
$.unmergeCells("A1:G10", true)

// 中文别名
$.z合并单元格("A1:G10", "cm")
$.z取消合并单元格("A1:G10", true)
```

## 📊 透视表结构公式

```
前导空白列数 = max(0, 行字段数 - 列字段数)
标题行数 = 列字段数 + 1
```

### 示例：双行字段 + 双列字段
```javascript
var result = Array2D.z超级透视(data, 'f1,f2', 'f3,f4', 'sum("f5")')
// 前导空白：1列（2-1=1，但实际是max(0,2-2)=0，要看实际输出）
// 标题行数：3行（2+1）
```

## ⚡ 常见问题

### Q: 合并没有效果？
A: 检查数据是否有连续相同的值

### Q: 取消合并后数据丢失？
A: 使用 `unmergeCells(rng, true)` 填充所有单元格

### Q: 如何编辑已合并的透视表？
A: 
```javascript
// 1. 取消合并
RngUtils.unmergeCells(rng, true)
// 2. 修改数据
Range("B1").Value2 = "2026"
// 3. 重新合并
RngUtils.mergeCells(rng, "cm")
```
