# superPivot 多层表头 - 修复前后对比

## 📊 核心问题对比

| 问题 | 修复前 (v3.9.0) | 修复后 (v3.9.1) |
|------|----------------|----------------|
| 前导空白列计算 | `numRowFieldLevels - 1` | `max(0, numRowFieldLevels - numColFieldLevels)` |
| 单列字段表头行数 | 3行 | 2行 |
| 多列字段表头行数 | `numColFieldLevels + 1` 行 | `numColFieldLevels + 1` 行 (保持不变) |
| 逻辑一致性 | 单列/多列不同逻辑 | 统一逻辑 |

---

## 🎨 具体场景对比

### 场景1: 单列字段 + 单行字段 (1行+1列)

#### 修复前 (❌ 错误)
```
Row 0: [产品] [2023] [2024]          ← 行字段标题 + 列值
Row 1: [年份] []     []              ← 列字段标题 + 空白
Row 2: [产品] [销售额] [销售额]      ← 行字段标题 + 数据标题 (❌重复)
```
**问题**: 3行表头，Row 0和Row 2重复了行字段标题

#### 修复后 (✅ 正确)
```
Row 0: [年份] [2023] [2024]          ← 列字段标题 + 列值
Row 1: [产品] [销售额] [销售额]      ← 行字段标题 + 数据标题
```
**正确**: 2行表头，结构清晰

---

### 场景2: 双列字段 + 单行字段 (1行+2列)

#### 修复前 (部分正确)
```
Row 0: [年份] [2024] [2024] [2025] [2025]  ← 列字段1标题 + 列值1
Row 1: [季度] [Q1]   [Q2]   [Q1]   [Q2]   ← 列字段2标题 + 列值2
Row 2: [产品] [销售额] [销售额] ...        ← 行字段标题 + 数据标题

Row 1: [季度] [Q1]   [Q2]   [Q1]   [Q2]   ← 列字段2标题 + 列值2
Row 2: [产品] [销售额] [销售额] ...        ← 行字段标题 + 数据标题
```

### 场景3: 单列字段 + 双行字段 (2行+1列)

#### 修复前 (❌ 前导空白错误)
```
Row 0: [] [年份] [2023] [2024]        ← 1个空白 (❌应该是0个)
Row 1: [类别] [产品] [销售额] [销售额] 
```
**问题**: 有1个前导空白，但 `max(0, 2-1) = 1`，这里刚好对了，但逻辑是错的

#### 修复后 (✅ 正确)
```
Row 0: [年份] [2023] [2024]          ← 0个空白 (max(0, 2-1) = 1? 等等)
Row 1: [类别] [产品] [销售额] [销售额]
```

**注意**: 让我重新计算前导空白：
- `max(0, 行字段数 - 列字段数) = max(0, 2-1) = 1`

所以修复后应该有1个前导空白，修复前有 `numRowFieldLevels - 1 = 1` 个，数量一样但计算方式不同。

但等等，根据维护说明文档，单列字段+双行字段的情况：
```
Row 0: 空值 + "年份" + 年份值
Row 1: "类别" + "产品" + 数据标题
```

确实有1个前导空白。让我再检查双列字段+双行字段的情况。

---

### 场景4: 双列字段 + 双行字段 (2行+2列)

#### 修复前 (❌ 前导空白错误)
```
Row 0: [] [年份] [2024] [2024] [2025] [2025]  ← 1个空白 (❌应该是0个)
Row 1: [] [季度] [Q1]   [Q2]   [Q1]   [Q2]   ← 1个空白 (❌应该是0个)
Row 2: [类别] [产品] [销售额] ...
```
**问题**: 有1个前导空白，但 `max(0, 2-2) = 0`，应该有0个

#### 修复后 (✅ 正确)
```
Row 0: [年份] [2024] [2024] [2025] [2025]  ← 0个空白 (max(0, 2-2) = 0)
Row 1: [季度] [Q1]   [Q2]   [Q1]   [Q2]   ← 0个空白
Row 2: [类别] [产品] [销售额] ...
```
**正确**: 行字段数=列字段数，不需要前导空白

---

### 场景5: 三行字段 + 双列字段 (3行+2列)

#### 修复前 (❌ 前导空白错误)
```
Row 0: [][] [大区] [2024] [2024] ...  ← 2个空白 (❌应该是1个)
Row 1: [][] [季度] [Q1]   [Q2]   ...  ← 2个空白 (❌应该是1个)
Row 2: [大区] [省份] [城市] [销售额] ...
```
**问题**: `numRowFieldLevels - 1 = 2`，但 `max(0, 3-2) = 1`

#### 修复后 (✅ 正确)
```
Row 0: [] [大区] [2024] [2024] ...  ← 1个空白 (max(0, 3-2) = 1)
Row 1: [] [季度] [Q1]   [Q2]   ...  ← 1个空白
Row 2: [大区] [省份] [城市] [销售额] ...
```
**正确**: 只有1个前导空白

## 🔧 关键代码对比

### 修复1: 表头行数计算

```javascript
// 修复前
var headerRowCount = (numColFieldLevels === 1) ? 3 : numColFieldLevels + 1;

// 修复后
var headerRowCount = numColFieldLevels + 1;
```

### 修复2: 前导空白列计算

```javascript
// 修复前
for (var rfIdx = 0; rfIdx < numRowFieldLevels - 1; rfIdx++) {
    headerRows[targetRow].push('');
}

// 修复后
var leadingBlankCols = Math.max(0, numRowFieldLevels - numColFieldLevels);
for (var i = 0; i < leadingBlankCols; i++) {
    headerRows[targetRow].push('');
}
```

---

## 📐 前导空白列数计算公式

```
leadingBlankCols = max(0, numRowFieldLevels - numColFieldLevels)
```

### 计算示例

| 行字段数 | 列字段数 | 公式 | 结果 |
|---------|---------|------|------|
| 1 | 1 | max(0, 1-1) | 0 |
| 1 | 2 | max(0, 1-2) | 0 |
| 2 | 1 | max(0, 2-1) | 1 |
| 2 | 2 | max(0, 2-2) | 0 |
| 3 | 1 | max(0, 3-1) | 2 |
| 3 | 2 | max(0, 3-2) | 1 |
| 3 | 3 | max(0, 3-3) | 0 |
