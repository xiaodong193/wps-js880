## 功能概述

`superPivot` (z超级透视) 是 JSA880 框架中最强大的数据透视功能，支持：

- **多级行字段**: 支持多个行字段分组（如：年份→季度→月份）
- **多级列字段**: 支持多个列字段分组
- **多种聚合**: 计数、求和、平均、最大、最小、文本连接
- **排序控制**: 支持升序(+), 降序(-), 自定义排序(#)
- **自定义标题**: 支持自定义字段标题
- **多种输出**: 数组、Map字典格式
- **表头控制**: 支持完整表头、无表头、不生成表头等多种选项
- **链式操作**: 支持筛选→透视、排序→透视等链式调用

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `arr` | Array | 必填 | 二维数组数据源 |
| `rowFields` | String/Array | 必填 | 行字段配置 |
| `colFields` | String/Array | 必填 | 列字段配置 |
| `dataFields` | String/Array | 必填 | 数据字段配置 |
| `headerRows` | Number | 1 | 源数据表头行数 |
| `outputHeader` | Number/String | 1 | 见下方详细说明 |

### outputHeader 参数详解

| 值 | 含义 | 说明 | 示例 |
|----|------|------|------|
| `1` 或 `true` | **完整表头** | 输出包含行标题和列标题的完整表头 | 默认模式 |
| `-1` | **表头不含行标题** | 输出表头，但不包含行标题列 | 适用于只需要列标题的场景 |
| `0` 或 `false` | **无表头** | 不输出表头，只有数据行 | 适用于数据导入等场景 |
| **`-2`** | **不生成表头** | 不生成表头（等同于0，但语义更明确） | ✨ v4.0 新增 |
| `'map'` | **Map模式** | 返回 Map 字典对象而非数组 | 便于进一步处理 |

## 常见问题

### Q1: outputHeader 各值有什么区别？

| 值 | 表头行数 | 行标题列 | 适用场景 |
|----|----------|----------|----------|
| `1` | 动态 | ✅ 有 | 默认，完整表头 |
| `-1` | 动态 | ❌ 无 | 只需要列标题 |
| `0` | 0 | ❌ 无 | 数据导入 |
| `-2` | 0 | ❌ 无 | 明确表示不需要表头 ✨ |

### Q2: 如何使用链式操作？

```javascript

arr.z筛选('f18 === "2023"').z超级透视(...)

// 排序 → 透视
arr.z多列排序('f2+,f26-').z超级透视(...)

// 筛选 → 排序 → 透视
arr.z筛选('f26 > 1000')
   .z多列排序('f2+,f3+,f26-')
   .z超级透视(...)
```

### Q3: 如何自定义字段标题？
