## 一、代码架构深度解析

### 1.1 寄生组合式继承（Parasitic Combination Inheritance）

**问题**: JavaScript中如何让自定义对象既拥有数组的特性，又能添加自定义方法？

**方案对比**:

| 方案 | 实现 | 优缺点 |
|------|------|--------|
| 原型继承 | `Array2D.prototype = new Array()` | 会执行Array构造函数，有问题 ❌ |
| 对象扩展 | 直接修改`Array.prototype` | 污染原生对象，极不推荐 ❌ |
| 包装模式 | 内部持有一个数组 | 需要代理所有数组方法，复杂且性能差 ❌ |
| **寄生组合式继承** | `Object.create(Array.prototype)` | 只继承原型，不执行父类构造函数 ✅ |

**实现代码详解**:

```javascript
// 【步骤1】创建一个以Array.prototype为原型的空对象
// Object.create()创建一个空对象，其[[Prototype]]指向传入的对象
Array2D.prototype = Object.create(Array.prototype);

// 【步骤2】修正constructor指向
// 上一步后constructor指向Array，需要手动修正回Array2D
Array2D.prototype.constructor = Array2D;

// 【步骤3】构造函数中填充数据
function Array2D(data) {
    // 支持工厂模式：允许不使用new调用
    if (!(this instanceof Array2D)) {
        return new Array2D(data);
    }
    
    // 数据规范化
    var items = [];
    if (data === null || data === undefined) {
        items = [];
    } else if (Array.isArray(data)) {
        items = data;
    } else {
        items = [[data]];  // 非数组包装为单元素二维数组
    }
    
    // 【关键】使用Array.prototype.push将数据附加到this
    // 这样this就成为真正的数组（具备length和索引访问）
    Array.prototype.push.apply(this, items);
}
```

**设计优点**:
1. Array2D实例是真正的数组（`[] instanceof Array === true`）
2. 可使用所有原生数组方法（`slice`, `concat`, `forEach`等）
3. `JSON.stringify`自动正确处理
4. `for...of`循环可用
5. 实例可以使用Array2D的自定义方法

### 1.2 链式调用设计模式

**核心原则**: 不可变性（Immutability）- 每个方法返回新实例，不修改原数据

**为什么使用不可变性？**
- 避免副作用，数据流清晰
- 易于调试，可以追踪每一步的中间状态
- 支持函数式编程风格

**实现核心：_new方法详解**:

```javascript
/**
 * 【链式调用核心】创建新Array2D实例
 * 
 * 技术流程：
 * 1. 创建空数组 instance = []
 * 2. 填充数据 Array.prototype.push.apply(instance, data)
 * 3. 设置原型 Object.setPrototypeOf(instance, Array2D.prototype)
 * 4. 添加内部属性（_original, _items）
 * 5. 返回instance
 */
Array2D.prototype._new = function(data) {
    // 创建空数组实例
    var instance = [];
    
    // 填充数据
    Array.prototype.push.apply(instance, data);

    // 设置原型链
    if (Object.setPrototypeOf) {
        // ES6标准方法
        Object.setPrototypeOf(instance, Array2D.prototype);
    } else {
        // 旧环境备用
        instance.__proto__ = Array2D.prototype;
    }

    // 添加内部属性
    Object.defineProperty(instance, '_original', {
        value: data,
        writable: true,
        enumerable: false,      // 不可枚举
        configurable: true
    });

    // _items访问器
    Object.defineProperty(instance, '_items', {
        get: function() {
            var arr = [];
            for (var i = 0; i < this.length; i++) {
                arr.push(this[i]);
            }
            return arr;  // 返回副本
        },
        set: function(value) {
            Array.prototype.splice.call(this, 0, this.length);
            Array.prototype.push.apply(this, value);
        },
        enumerable: false,
        configurable: true
    });

    return instance;
};
```

**链式调用示例**:

```javascript
Array2D.prototype.skip = function(count) {
    // 返回新实例，原数组不变
    return this._new(this._items.slice(count));
};

// 使用
var result = arr.skip(3).take(10).filter('f1>0');
// 等价于：
// var step1 = arr.skip(3);
// var step2 = step1.take(10);
// var result = step2.filter('f1>0');
```

### 1.3 Lambda表达式解析器

**支持的表达式形式**:

| 形式 | 示例 | 转换结果 | 适用场景 |
|------|------|----------|----------|
| 列选择器 | `'f1>0'` | `_ => _[0]>0` | 简单列操作 |
| 索引选择器 | `'$0*2'` | `_ => _[0]*2` | 数组索引访问 |
| 箭头函数 | `'row=>row[0]>0'` | 直接使用 | 复杂逻辑 |
| 复杂表达式 | `'f1+f2>100'` | `_ => _[0]+_[1]>100` | 多列计算 |

**解析流程详解**:

```javascript
function parseLambda(expr) {
    // 【步骤1】缓存检查
    // 使用对象作为缓存，避免重复解析同一表达式
    if (_lambdaCache[expr]) return _lambdaCache[expr];

    var fn;
    try {
        // 【步骤2】检测箭头函数语法
        // 包含'=>'的表达式直接作为箭头函数处理
        if (LAMBDA_PATTERNS.ARROW_FUNCTION.test(expr)) {
            fn = eval('(' + expr + ')');
        }
        // 【步骤3】检测$索引语法
        // 将$0, $1转换为_[0], _[1]
        else if (expr.includes('$')) {
            const indexMatch = expr.match(LAMBDA_PATTERNS.INDEX_SELECTOR);
            if (indexMatch) {
                // 安全检查：防止索引越界
                const maxIndex = Math.max(...indices);
                if (maxIndex > ARRAY_LIMITS.MAX_INDEX) {
                    console.warn('Lambda索引超出限制:', maxIndex);
                    return null;
                }
                // 转换并创建函数
                fn = new Function('_', 'return ' + 
                    expr.replace(LAMBDA_PATTERNS.INDEX_SELECTOR, '_[$1]'));
            }
        }
        // 【步骤4】检测f列选择器语法
        // 将f1, f2转换为_[0], _[1]（注意f1对应索引0）
        else if (LAMBDA_PATTERNS.COLUMN_SELECTOR.test(expr)) {
            fn = new Function('_', 'return ' + 
                expr.replace(/f(\d+)/g, '_[$1-1]'));
        }
        // 【步骤5】其他情况作为表达式
        else {
            fn = new Function('_', 'return ' + expr);
        }
    } catch (e) {
        console.warn('Lambda解析失败:', expr, e);
        return null;
    }

    // 缓存并返回
    _lambdaCache[expr] = fn;
    return fn;
}
```

## 三、$ 符号功能

### 3.1 基本用法

`$` 是 JSA880 的核心快捷符号，提供多种功能：

```javascript
// 单元格快捷访问
$("A1")              // 等同于 Range("A1")
$(5, 1)             // 第5行第1列
$("A1").z值()        // 获取值

// 工具类快捷访问
$.maxArray("A1:H1");  // 获取最大行区域并转为数组
$.safeArray("A1:D10"); // 安全获取区域数组
$.endRow("A:A");       // 获取最大行数
```

### 3.2 $.maxArray 详解

```javascript
/**
 * 获取从第一行到最大行的区域并转换为 Array2D 对象
 * @param {Range|string} rng - 起始区域（如 "A1:H1"）
 * @param {string} [col="A"] - 列号（整行时需要）
 * @returns {Array2D} Array2D 对象，支持链式调用
 */

// 基本用法
var arr = $.maxArray("A1:H1");

// 链式调用示例
$.maxArray("A1:H1")
  .skip(3)              // 跳过前3行
  .take(10)             // 取前10行
  .sortByCols('f2')     // 按第2列排序
  .filter('f6==2023')   // 筛选
  .toRange("K4");       // 输出到K4
```

## 十一、链式调用最佳实践

### 11.1 推荐写法

```javascript
// 流畅的链式调用
$.maxArray("A1:H1")
    .skip(1)
    .filter('f2>0')
    .sortByCols('f1+,f2-')
    .take(100)
    .toRange("K1");

// 分步调试
var arr = $.maxArray("A1:H1");
var filtered = asArray2D(arr).filter('f2>0');
var sorted = filtered.sortByCols('f1+');
var top100 = sorted.take(100);
top100.toRange("K1");
```

### 11.2 注意事项

```javascript
// 静态方法返回普通数组
var arr = [[1,2], [3,4], [5,6]];
var rs = Array2D.skip(arr, 1);  // 返回普通数组
rs.toRange("A1");               // 普通数组也有 toRange 方法

// 实例方法返回 Array2D 对象
var arr = Array2D([[1,2], [3,4]]);
var rs = arr.skip(1);           // 返回 Array2D
rs.toRange("A1");
```

## 十二、常见问题

### Q1: $.maxArray 和 $.safeArray 有什么区别？

```javascript
// $.maxArray - 从指定行扩展到最大行
var arr = $.maxArray("A1:H1");

// $.safeArray - 获取指定区域的数组
```

### Q2: 如何处理表头？

```javascript
// 方法1：skip 跳过
$.maxArray("A1:H1").skip(1).toRange("K1");

// 方法2：sortByCols 的 skipHeader 参数
$.maxArray("A1:H1").sortByCols('f2+', 1).toRange("K1");
```

### Q3: 中文方法名和英文方法名的区别？

```javascript
// 两者完全等价，可以混用
asArray2D(arr)
    .skip(3)
    .z取前N个(10)
    .sortByCols('f1+')
    .z筛选('f2>0')
    .toRange("A1");
```

## 十二、超级透视后合并单元格的正确方法

### 12.1 问题描述

在使用 `超级透视` 后，尝试合并行字段或列字段标题时，如果使用了错误的方法，会导致**整列或整行被合并**：

```javascript
// ❌ 错误代码 - 会导致整列/整行被合并
var rng = rs.getRange("k2");
$.mergeCells($(rng).Columns("A:B"));      // 错误：合并了整个工作表的A、B列
$.mergeCells($(rng).Rows("1:2"), 'c');    // 错误：合并了整个工作表的前2行
```

### 12.2 问题原因

```
问题分析流程：
1. rs.getRange("k2") 返回 WPS Range 对象（如 K2:Z20）
2. $(rng) 将 Range 包装成 RangeChain 对象
3. $(rng).Columns("A:B") 返回 WPS 原生 Columns 集合
   ⚠️ 注意：Columns("A:B") 引用的是【整个工作表的A、B列】
4. $.mergeCells 直接对整个 Columns 执行合并
```

### 12.3 正确解决方案

#### 方案1：使用 Resize 限定范围（推荐）

```javascript
function 超级透视_test无表头() {
    清空数据();
    
    var arr = $.maxArray("数据!A2:H2");
    arr = Array2D.sortByCols(arr, 'f2,f3,f6,f7', 0);
    arr = Array2D.filter(arr, "f7>=1 && f7<=3", 0);
    
    var rs = Array2D.superPivot(arr,
        ['f2,f3', '产品,国家'],
        ['f6+,f7-', '年,月'],
        ['count(),sum("f4"),average("f5")'],
        0);
    
    rs.toRange('K2');
    var rng = rs.getRange("k2");
    
    // ✅ 正确：使用 Resize 限定范围
    var rowFieldsCount = 2;  // 行字段数量（产品、国家）
    var dataRowCount = rng.Rows.Count - 2;  // 数据行数（减去表头）
    
    // 合并行字段列（只合并数据区域，不包括表头）
    for (var i = 1; i <= rowFieldsCount; i++) {
        var colRange = rng.Columns(i).Resize(dataRowCount);
        $.mergeCells(colRange);
    }
    
    // 合并列字段标题行
    var colFieldsCount = 2;  // 列字段层级（年、月）
    for (var r = 1; r <= colFieldsCount; r++) {
        var startCol = rowFieldsCount + 1;
        var endCol = rng.Columns.Count;
        var rowRange = rng.Range(
            rng.Cells(r, startCol),
            rng.Cells(r, endCol)
        );
        $.mergeCells(rowRange);
    }
    
    $.addBorders(rng);
    rng.Columns.AutoFit();
}
```

#### 方案2：使用 Range 方法指定相对区域

```javascript
function 超级透视_test无表头_方案2() {
    清空数据();
    
    var arr = $.maxArray("数据!A2:H2");
    arr = Array2D.sortByCols(arr, 'f2,f3,f6,f7', 0);
    arr = Array2D.filter(arr, "f7>=1 && f7<=3", 0);
    
    var rs = Array2D.superPivot(arr,
        ['f2,f3', '产品,国家'],
        ['f6+,f7-', '年,月'],
        ['count(),sum("f4"),average("f5")'],
        0);
    
    rs.toRange('K2');
    var rng = rs.getRange("k2");
    
    // ✅ 正确：使用 Range 方法获取相对区域
    var rowFieldsCount = 2;
    var colFieldsCount = 2;
    
    // 合并行字段（从第3行开始，因为前2行是列标题）
    for (var c = 1; c <= rowFieldsCount; c++) {
        var startCell = rng.Cells(3, c);  // 第3行开始
        var endCell = rng.Cells(rng.Rows.Count, c);
        $.mergeCells(rng.Range(startCell, endCell));
    }
    
    // 合并列字段标题
    for (var r = 1; r <= colFieldsCount; r++) {
        var startCell = rng.Cells(r, rowFieldsCount + 1);
        var endCell = rng.Cells(r, rng.Columns.Count);
        $.mergeCells(rng.Range(startCell, endCell));
    }
    
    $.addBorders(rng);
    rng.Columns.AutoFit();
}
```

### 16.2 模块加载顺序问题

#### 问题描述
JSA 中多个模块同时加载时，可能出现变量未定义的引用错误。

#### 根本原因
1. JSA 的模块加载顺序与 JavaScript 执行顺序不同
2. IIFE 中的代码在模块加载时立即执行
3. 跨模块的变量引用可能在目标模块加载前执行

#### 通用解决方案
```javascript
// ❌ 错误: 直接引用 (模块加载时执行)
this.RngUtils = RngUtils;  // RngUtils 可能未定义

// ✅ 正确: 延迟执行 (函数调用时执行)
this.getRngUtils = function() {
    if (typeof RngUtils !== 'undefined') {
        return RngUtils;
    }
    return null;
};

// ✅ 正确: 函数表达式包装
this.safeArray = function(rng) {
    if (typeof RngUtils !== 'undefined' && typeof RngUtils.safeArray === 'function') {
        return RngUtils.safeArray(rng);
    }
    // fallback logic
};
```
