## 核心数据结构

### 1. 分组映射 (groupMap)

```javascript
// 键格式: "行键|||列键"
// 值格式: [数据行1, 数据行2, ...]
groupMap = {
    "产品A|||中国": [[row1], [row2], ...],
    "产品A|||美国": [[row3], [row4], ...],
    ...
}
```

### 2. 行键/列键

```javascript
// 多级字段用 @^@ 分隔
rowKeys = [
    "产品A",                    // 单级
    "产品A@^@北京",             // 多级行键
    "产品A@^@北京@^@2024"       // 三级行键
]

colKeys = [
    "2024",                     // 单级
    "2024@^@Q1",               // 多级列键
    "2024@^@Q1@^@1月"          // 三级列键
]
```

### 3. 字段配置对象

```javascript
{
    fields: [
        { field: 'f1', sort: '+' },   // 字段名 + 排序符号
        { field: 'f2', sort: '-' }
```
