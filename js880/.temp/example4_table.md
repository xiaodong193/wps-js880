#### 合并规则详解
```
列标题合并（横向）:
  Row 0:
    - A0:A0 → 空
    - B0:B0 → "年份"单独，不合并
    - C0:F0合并 → "2024"连续4次
    - G0:H0合并 → "2025"连续2次

  Row 1:
    - A1:A1 → 空
    - B1:B1 → "季度"单独，不合并
    - C1:D1合并 → "Q1"连续2次
    - E1:F1合并 → "Q2"连续2次
    - G1:G1 → "Q1"单独（不与H1合并，因为H1是"Q2"）
    - H1:H1 → "Q2"单独

  Row 2: 不合并（数据标题行）

行标题合并（纵向）:
  第一列（类别）:
    - A3:A7合并 → "电子产品"连续5次（包含3个手机+2个电脑）
    - A8:A9合并 → "家电产品"连续2次

  第二列（产品）:
    - B3:B5合并 → "手机"连续3次
    - B6:B7合并 → "电脑"连续2次
    - B8:B9合并 → "冰箱"连续2次
```
