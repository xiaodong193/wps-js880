# KO一切的k函数 · WPS UDF 模块

> ⚠️ 本模块已于 v4.2.2 完整合并到 `js880/JSA880.js` 中
> 
> **不再需要单独加载！** 顶层 `function k()` / `function jsaLambda()` 现在直接定义在主框架文件里。

## 使用方法(单文件部署)

```bash
# 方式 A(推荐):作为 WPS 加载项
WPS → 选项 → 加载项 → 加载 JSA880.js
# 之后所有 WPS 文档的单元格都能用 =k(...) 公式

# 方式 B:JSA 编辑器
打开 WPS → 开发工具 → JSA 编辑器 → 新建模块 → 粘入 JSA880.js 全文
# 保存后,Console 会看到"✅ k() UDF 已就绪!"
```
