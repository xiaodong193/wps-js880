## 🐛 常见问题速查

| 问题 | 解决方法 |
|------|----------|
| Array2D 未定义 | 加载 JSA880.js |
| 测试数据为空 | 检查 createTestData() |
| 函数报错 | 检查参数格式 |
| 测试卡死 | 按 Esc 停止，减少数据量 |
| 版本不对 | 更新到 v3.9.0 |
