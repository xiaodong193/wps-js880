## 🐛 常见问题

### Q1: 无法访问 VBA 项目
**错误**: "程序matic access to Visual Basic Project is not trusted"

**解决方法**:
1. Excel → 文件 → 选项 → 信任中心
2. 点击"信任中心设置"
3. 左侧选择"宏设置"
4. 勾选"信任对 VBA 工程对象模型的访问"
5. 确认并重启 Excel

### Q2: 模块名称冲突
**错误**: 模块名称已存在

**解决方法**:
1. 删除旧模块
2. 或使用不同的名称（如 `TestModule2`）

### Q3: 代码太长，粘贴卡顿
**解决方法**:
1. 分段复制粘贴
2. 或先保存到文件，再导入

### Q4: Mac 上的差异
**Mac 快捷键**:
- `Fn + Alt + F11` - 打开 VBA 编辑器
- `Cmd + G` - 打开立即窗口
- `Cmd + S` - 保存
